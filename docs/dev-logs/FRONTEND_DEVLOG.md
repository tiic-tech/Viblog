## UI/UX Guidelines

### Design System
- 8px grid system
- Border radius: 8-12px
- Section gaps: 60-80px
- One accent color
- Micro-interactions: 200ms ease

### Accessibility
- WCAG 2.1 AA compliance
- Keyboard navigation
- Screen reader support
- Color contrast 4.5:1 minimum

## Real-World Testing Requirements

Before any UI component is marked complete:

```
□ Loaded in actual browser
□ All interactive elements clicked
□ Responsive breakpoints tested
□ Hover/focus states verified
□ Loading states tested
□ Error states tested
□ User flow end-to-end
□ Accessibility checked
```
